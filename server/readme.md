# Film Vote Server

#### Description

An open source app to let you and your friends vote on the new films to watch.

#### Build / Run

Gradle

```
gradle clean bootRun
```

Maven

```
mvn clean install spring-boot:run
```

#### Interface

[Server API via Swagger UI](http://localhost:8080/swagger-ui.html)
